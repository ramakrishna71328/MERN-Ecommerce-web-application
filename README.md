# MERN E-Commerce Web Application

## Overview

A full-stack E-Commerce web application developed using the MERN Stack (MongoDB, Express.js, React.js, Node.js). The application allows users to browse products, manage shopping carts, authenticate securely, and place orders through a responsive interface.

## Features

* User Registration and Login
* Secure Authentication
* Product Listing and Search
* Product Details Page
* Shopping Cart Management
* Responsive User Interface
* REST API Integration
